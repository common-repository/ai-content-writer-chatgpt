== Description ==

AI Content Writer - ChatGPT is a content generator package for WordPress. It is the most popular, WordPress-based open-source AI solution. It utilizes GPT-3.5(https://openai.com) davinci(model) and other models to generate content. Provides features for very quick use of prompt and built-in forms. Very easy to set Open AI parameters and very quick to get content from Open AI.Also, the facility to choose a writing style and writing tone for content. Ready-to-use AI forms and prompts can use directly and also from the shortcode. Also stored recent prompt responses and forms response. Also provides categories for AI forms and prompts so easily can find. The posts and pages section also gives "OpenAi Content Generator" button to easily generate title and content.

== Installation ==
1. Install AI Content Writer - ChatGPT from the WordPress repository or upload the plugin files to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Setup OpenAI API key.
4. Access the settings page from the 'AI Content Writer - ChatGPT' menu in WordPress admin dashboard and configure the other options(ie.temperature,max tokens etc).
5. Start generating content through AI Content Writer - ChatGPT which is use OpenAI GPT-3.5, davinci(model) to generate content.

== How to Generate OpenAI API Key ==
1.Go to OpenAI's Platform website at https://platform.openai.com and sign in with an OpenAI account.
2.Click your profile icon at the top-right corner of the page and select "View API Keys."
3.Click "Create New Secret Key" to generate a new API key.
